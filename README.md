# 📊 Call Center Analytics Dashboard 

## 📌 Overview

This Power BI project provides an interactive dashboard for analyzing call center performance. It enables stakeholders to monitor call volume, response efficiency, customer sentiment, and operational distribution across channels, locations, and time.

---

## 🎯 Objectives

* Track total call activity and trends over time
* Monitor response time and service efficiency
* Analyze customer sentiment distribution
* Identify performance by state, channel, and call center
* Enable data-driven decision-making

---

## 📈 Key Metrics

* **Total Calls:** 18.77K
* **Total Call Duration (Count):** 7.82K
* **Total Call Duration (Time):** 468.99K
* **Average Call Duration:** 24.99
* **Response Time %:** 75.23%

---

## 📊 Dashboard Features

### 1. Call Trends

* Total calls by day to identify peak periods
* Helps in workforce planning

### 2. Geographic Insights

* Calls distribution by state (map visualization)
* Highlights regional demand

### 3. Call Reasons

* Breakdown of calls (e.g., Billing, Payments, Service)
* Identifies common customer issues

### 4. Channel Analysis

* Distribution across:

  * Call Center
  * Chatbot
  * Email
  * Web

### 5. Sentiment Analysis

* Categorized into:

  * Negative
  * Neutral
  * Positive
  * Very Positive/Negative

### 6. Call Center Performance

* Comparison of call volumes across centers
* Identifies top and underperforming locations
